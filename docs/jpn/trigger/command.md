# コマンドトリガー
**コマンドを実行した時に発火します．**

以下の変数が追加されます

| 変数名 | 説明                                                         | 型 |
| ---- |------------------------------------------------------------| ---- |
| cmd | コマンドを空白で区切った最初の部分                                          | [string](/variable/types.md#string) |
| args | コマンドを空白で区切った2つ目以降(`/warp aieuo` を実行した時、 `{args.0}` はaieuo) | [list [string]](/variable/types.md#list) |
